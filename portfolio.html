Coding a image generator

In the summer of 2024, I wanted to finish an art project I'd been scheming. After thinking of all it's possible iterations, I decided it would be more efficient to code the creation process. To fill a 8x8 grid of squares with 4 different colors in four mirrored quadrants makes 13,824 entries. I only did this math after finishing a python program that prints raw RGB data into a ppm file. I didn't want to wait for my computer to process all these iterations, so I cut my losses and came out with the simple image generator.
